ES6 Notes

I - let và var khác nhau:

  • 1. Khai báo
    • - var thì cho khai báo biến giống nhau trong cùng 1 khu vực.
    • - let thì không cho khai báo biến giống nhau trong cùng 1 1 khu vực
  • 2. Phạm vi sử dụng:
    • - let chỉ cho sử dụng trong 1 block (cặp ngoặc {})
    • - var thì cho sử dụng bên ngoài 1 block (cặp ngoặc {})
  • 3. hoisting (là cơ chế thu thập biến trước khi chạy code. Nghĩa là khai báo biến trước hay sau cũng được)
    • - let không hỗ trợ hoisting
    • - var hỗ trợ hoisting

    • II - Arrow function

    const calsSum = () => 1 + 1; const showObject = () =>({name: "hieu"}) const testArrow = function(){ console.log(this) }
    Khác nhau Arrow function và fucntion bình thường

    Function thường làm thay đổi ngữ cảnh của this.

    Arrow function không làm thay đổi ngữ cảnh của this

    Do vậy nếu function có từ khóa this thì phải để ý tới do đó trong class thì không nên dùng Arrow Function
