Thẻ a liên kết - anchor

  • Cú pháp
    • <a href="URL" target="">Linked content</a>
  • Thuộc tính "target" của thẻ a
    • name: tải trang web vào frame có tên NAME
    • _blank:tải trang web vào cửa sổ mới
    • _parent:tải trang web vào cửa sổ cha của nó
    • _sel:tải trang web vào cửa sổ hiện hành
    • _top: tải trang web vào cửa sổ cao nhất
  • Ví dụ
    • Yahoo

Phân loại:

  • Liên kết ngoại (external)
    • <a href="url"> External link</a>
  • Liên kết nội (internal)
    • Liên kết trong cùng một trang ví dụ:
    • <a name="TenViTri"> Vị trí bắt đầu</a> <a href="#TenViTri"> Text đại diện</a>
  • Liên kết email (email)
    • <a href="mailto:emailAddress">Liên hệ Admin</a>
    • Liên hệ Admin

Thẻ liên kết trang

- URL chia làm 2 loại

  • Địa chỉ tuyệt đối so với mạng internet
  • Địa chỉ tương đối so với trang web hiện hành đang chứa liên kết

- Một số ký hiệu đường dẫn đặc biệt

/: Trở về thư mục gốc của website ./: Thư mục hiện tại của trang web sử dụng link (mặc định) ../: Quay ra thư mục cha / đi ngược lại 1 cấp thư mục
Ký hiệu Ý nghĩa
/ Trở về thư mục gốc của website
./ Thư mục hiện tại của trang web sử dụng link (mặc định)
../ Quay ra thư mục cha / đi ngược lại 1 cấp thư mục
