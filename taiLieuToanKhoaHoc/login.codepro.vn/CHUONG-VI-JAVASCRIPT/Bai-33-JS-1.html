Script là gì

  • Client-side script
    • Script được thực thi tại Client-Side (trình duyệt): Thực hiện các tương tác với người dùng ( tạo menu chuyển động...) kiểm tra nhập liệu
  • Server-Side Script
    • Script được xử lý tại Server-Side, nhằm tạo các trang web có khả năng phát sinh nội dung động. Một số xử lý chính: Kết nối CSDL, truy cập hệ thống file trên server, phát sinh nội dung html trả về người dùng...

JavaScript là gì

Không phải là ngôn ngữ của Java

  • JavaScript là ngôn ngữ Client-side script hoạt động trên trình duyệt của người dùng (client)
  • Chia sẻ xử lý trong ứng dụng web. Giảm các xử lý không cần thiết trên server
  • Giúp tạo các hiệu ứng tương tác cho trang web

Nhúng JavaScript vào trong HTML

  • Nhúng vào trang web với tag script <script>//Lệnh Javascript</script>
  • Nhúng vào từ file bên ngoài <script src="main.js"></script>
  • Web browser sẽ thực thi script theo thứ tự từ trên xuống dưới
  • Đối với external source code JavaScript có thể đặt trong các file .js sẽ được nhúng vào HTML trước khi hoạt động
  • các đoạn code JavaScript được browser xử lý cùng thứ tự với các thẻ HTML. Trừ các hàm (function) chỉ được thực hiện khi có lời gọi hàm
  • Chèn JavaScript trong thẻ body hoặc trong thẻ head đều được
    • Nếu đặt giữa tag Head thì script sẽ thực hiện ngay khi mở web
    • Nếu đặt giữa tab body thì script được thực thi khi trang web đang mở
  • Số lượng script đặt trong website không hạn chế
