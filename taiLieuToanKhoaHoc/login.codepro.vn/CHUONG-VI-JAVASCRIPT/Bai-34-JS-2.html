Biến số trong JavaScript

  • Cách đặt tên biến
    • Bắt đầu bằng chữ cái hoặc dấu_
    • A...Z, a...z, 0...9,_: phân biệt HOA, thường
    • Ví dụ: var chuoiSo var ChuoiSo
  • Khai báo biến
    • Sử dụng từ khóa var
    • Ví dụ var count = 10, amount;
    • Không cần khai báo biến trước khi sử dụng, biến thật sự tồn tại khi bắt đầu sử dụng lần đầu tiên
    • Biến không cần khai báo kiểu dữ liệu vì biến trong JavaScript không có kiểu dữ liệu nhất định
  • Kiểu dữ liệu
    • Kiểu dữ liệu Ví dụ Mô tả
    Object var listBooks = new Array (10) Trước khi sử dụng phải cấp phát bằng từ khóa new
    String "The cow jum.. "40" Chứa được chuỗi unicode, chuỗi rỗng ""
    Number 0.99000 or 12 Theo chuẩn IEEE 754
    boolean true/false
    undefined var myVariable myVariable = undefined
    null connection.Close(); connection = null

  • Đổi kiểu dữ liệu
    • Biến tự đổi kiểu dữ liệu khi giá trị mà nó lưu trữ thay đổi Ví dụ: var x = 10; // x kiểu Number x ="Hello world!"; // x kiểu String
    • Có thể cộng 2 biến khác kiểu dữ liệu

      • Ví dụ

      var x; x = "12" + 34.5 // KQ x = "1234.5"

    • Hàm parseInt(...), parseFloat(...): Đổi kiểu dữ liệu từ chuỗi sang số
  • Hàm trong JavaScript

    • Là tập hợp các lệnh có dạng thức khai báo chung:

      function Tên_hàm (thamso1, thamso2...) { // lệnh của hàm }

    • Hàm có giá trị trả về

      • Ví dụ

      function Tên_hàm (thamso1, thamso2, ...) { return (value); }

    Ví dụ hàm trong JavaScript

    function Sum(x,y) { tong = x + y; return tong; }

    Gọi hàm

    var x = Sum(10,20);

    hiển thị xuất ra kết quả hàm

    alert(x);
